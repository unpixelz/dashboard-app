# Trading Dashboard – Lernprojekt

Ein **privates Next.js-Projekt** zum Aufbau eines modernen Trading-Dashboards mit Authentifizierung und Datenbank-Anbindung.

---

## Tech Stack

* **Next.js** (App Router) + **TypeScript**
* **Supabase** (Auth & Database)
* **Tailwind CSS**
* **React**

---

## Features

* Authentifizierung (Login/Signup)
* Trading Dashboard mit Live-Daten
* Dark Mode
* Supabase-Integration

---

## Installation & Start
```bash
npm install
npm run dev
```

---

## Ziele

* Next.js App Router & Server Components
* Supabase Auth & DB Anbindung
* Saubere Projektarchitektur
* Trading-UI/UX Graph Data
* Trade Journaling

---

**Hinweis:** Rein privates Lernprojekt – nicht produktionsreif.
